# 名字
Astesola，星空。  
名字的Aste和sola都来自拉丁语，分别是"星星"的前缀和"孤独"的意思。同时"sola"也与日语中"空(そら)"的读音相同。
# 个人爱好
星空是个生活在网络上的人，线下极度社恐；   
热爱音乐、是个深度术术人，听的歌绝大部分是V曲； 
重度白河豚，平时看的ACGN作品多以百合题材为主； 
喜欢一切萌系的事物，最喜欢的画师是[しらたま](https://twitter.com/shiratamacaron)老师； 
梦想是成为一体机P主；
